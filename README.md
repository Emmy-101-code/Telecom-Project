EmmyPay Telecom - Vite + React + Tailwind starter

Quick start:
  1. npm install
     If you see issues with tailwind, run:
       npm config set registry https://registry.npmjs.org/
       npm install
  2. npm run dev
  3. Open http://localhost:5173

Notes:
- Email link in Contact uses mailto: to open user's mail client (Gmail will open if configured).
- Replace placeholder images in /src/assets with your preferred photos.
